# Script — Codex Micro (written to frames v1)

9 segments, ~160 words ≈ ~58s voice / ~64s reel. Register: brisk launch-day
energy, playful but factual. Hook archetype: **contrarian/overlooked** ("first
hardware is… a keyboard?") — differs from neutrality (contrarian-result) and
GRAM (number-shock) by subverting the expected *object*, not a stat.

## V1 → critique

V1 opened "OpenAI just launched new hardware called Codex Micro, a macro pad
for agentic coding workflows." Critique: (i) "macro pad" and "agentic coding
workflows" are off-table jargon; (ii) no gloss on OpenAI or Codex for a
zero-background viewer; (iii) "$6.5B acquisition of io" stated as bare fact —
research marks it *reported*; (iv) hook buried the surprise (keyboard) behind
the product name. V2 fixes all four with the translation table's plain forms.

## Comprehension-gate iteration log

- **Iteration 1: FAIL.** Fresh viewer flagged (i) "secret device from the
  iPhone's designer" — an insider reference with no setup (the $6.5B landed
  "on a mystery"); (ii) ChatGPT used as an identifier but never defined;
  (iii) the jump from ONE Codex to multiple "AI workers" forced a guess. All
  script-clarity defects → fr7 now sets up the io beat in-line ("bought a
  secret gadget company from the iPhone's designer"), fr1 defines ChatGPT by
  category ("the famous AI chatbot"), fr4 states multiplicity ("it can run
  several jobs at once").
- **Iteration 2: PASS.** Correct retell, every term/number defined from the
  narration alone, (d) NONE — the io contrast now "immediately answered its
  own question."
